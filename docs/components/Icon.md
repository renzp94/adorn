# Icon 图标

## 基本使用

通过指定`name`指定图标，通过`size`属性指定大小，通过`color`指定颜色

<demo src="../../../../example/icon/index.svelte" github="Icon"></demo>


## API

### Props 属性

| 属性 | 说明     | 类型     | 默认值 | 版本 |
| ---- | -------- | -------- | ------ | ---- |
| name | 图标名   | `string` | -      |      |
| size | 图标大小 | `string` | `14px` |      |
